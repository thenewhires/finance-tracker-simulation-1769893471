# Running the application locally

**Backend (Java Spring):**

1.  Navigate to the `backend` directory.
2.  Run `mvn spring-boot:run` to start the backend server.

**Frontend (Vue):**

1.  Navigate to the `frontend` directory.
2.  Run `npm install` to install dependencies.
3.  Run `npm run serve` to start the frontend development server.

Make sure the backend server is running before starting the frontend.

The frontend will be available at `http://localhost:8080` (or the configured port in `vue.config.js`). The backend will run on `http://localhost:8081`.
